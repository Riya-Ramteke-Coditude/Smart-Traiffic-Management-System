# Smart Traffic Management System 🚦

## 📌 Overview

This project is an AI-based Smart Traffic Management System that helps in optimizing traffic flow using object detection techniques.

## 🚀 Features

* Vehicle detection using YOLO
* Traffic density analysis
* Smart signal control

## 🛠️ Technologies Used

* Python
* OpenCV
* YOLOv8
* NumPy

## 📂 Project Structure

* MRSAC_Intern.ipynb

## ▶️ How to Run

1. Open the notebook in Google Colab
2. Run all cells
3. Upload an image when prompted
4. The system will detect vehicles and analyze traffic

## 📸 Input

* User uploads an image manually for detection

## 📈 Future Scope

* Real-time traffic control
* Integration with live CCTV feeds
* Automation using IoT
